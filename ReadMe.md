This is a basic webpack 4 boilerplate to work with typescript and sass.


## Installing the starter kit
 
 > Go to your folder and clone : git clone https://github.com/samitrimal/webpack-typescsript-sass-starter.git
 > Run ``` npm install ```
 > Run ``` npm init ```
